# Design Studio Lead Engine

Проект для системы лидогенерации студии проектирования.

## Цель

Построить систему, которая:

- находит источники лидов;
- собирает сигналы спроса на проектирование;
- очищает и обогащает лиды;
- расставляет приоритет;
- отправляет результат в работу автоматически или полуавтоматически.

## Что важно

- **Два направления:** инженерное проектирование (вся Россия) + перепланировки (вся Россия кроме Москвы и МО)
- **Архитектура:** одна мультиагентная система, где агенты параллельно собирают лиды из разных источников и передают в единый пайплайн
- **CRM:** Bitrix24 — конечная точка маршрутизации
- **Связанный проект:** KP-генератор (`/идея проекта КП/`) — lead engine кормит его
- Сначала умный MVP на одном потоке, потом масштабирование
