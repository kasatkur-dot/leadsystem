## 1. Что делаем с этой информацией

SignalCard нужны не для хранения ссылок, а для превращения рыночных сигналов в маленькие проверяемые действия.

Правильная цепочка:

```text
SignalCard -> маленькая задача -> ответственный агент -> проверка -> status used/rejected
```

На этот шаг выбраны 3 карточки:

- `SignalCard 01` — для сайта;
- `SignalCard 07` — для контента;
- `SignalCard 03` — для лидогенерации/CRM.
