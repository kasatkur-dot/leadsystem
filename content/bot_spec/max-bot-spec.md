# Техническое задание

## Статус

Этап открыт. Дата: 2026-04-24.

---

## Что создаём

Систему из трёх независимых частей:

1. **Бот в MAX** — умный автоответчик на слово ПРОВЕРКА с ИИ-ответом через Gemini
2. **Канал в MAX** — правильно настроенный, с автопостингом по расписанию
3. **Точки входа** — карты, Авито, SEO внутри MAX

Всё бесплатно, без платных подписок.

---

## Часть 1. Бот

### 1.1 Что делает бот

- Ловит слово ПРОВЕРКА в личных сообщениях
- Задаёт 3 уточняющих вопроса с кнопками
- Отправляет ответы в Gemini и получает умный персональный ответ
- Отправляет этот ответ клиенту
- Уведомляет эксперта с готовыми данными клиента

### 1.2 Полный сценарий диалога

```
Клиент → боту: «ПРОВЕРКА» (или «проверка», регистр не важен)

Бот → клиенту:
«Хорошо, разберём вашу ситуацию.
Ответьте на 3 коротких вопроса — это займёт минуту.»

Бот → клиенту (Вопрос 1):
«Из какого вы города?»
[Краснодар] [Краснодарский край] [Другой город]

Клиент → боту: нажимает кнопку (или пишет текстом)

Бот → клиенту (Вопрос 2):
«Что сейчас происходит с вашей перепланировкой?»
[Планирую ремонт, хочу сделать по закону]
[Уже сделал без документов]
[Покупаю или продаю квартиру]
[Банк требует документы по ипотеке]

Клиент → боту: нажимает кнопку

Бот → клиенту (Вопрос 3):
«Есть план БТИ или технический паспорт квартиры?»
[Да, есть] [Нет] [Не знаю]

Клиент → боту: нажимает кнопку

→ Бот собирает все три ответа
→ Отправляет запрос в Gemini API
→ Получает умный ответ
→ Отправляет ответ клиенту
→ Уведомляет эксперта

Бот → клиенту (умный ответ от Gemini):
[персональный текст на основе ситуации клиента]
«Эксперт свяжется с вами в течение нескольких часов.»

Бот → эксперту (уведомление в MAX):
«🔔 Новый запрос ПРОВЕРКА
Город: [ответ]
Ситуация: [ответ]
БТИ: [ответ]
Написал: [имя клиента в MAX]»
```

### 1.3 Промпт для Gemini

Шаблон, который бот отправляет в Gemini после сбора данных:

```
Ты — вежливый помощник инженерно-проектной компании,
которая занимается согласованием перепланировок квартир.

Данные клиента:
- Город: {город}
- Ситуация: {ситуация}
- Наличие плана БТИ: {бти}

Напиши короткий (3–5 предложений) тёплый ответ от лица эксперта.
Кратко обозначь, с чем скорее всего столкнётся клиент в его ситуации
и что важно проверить. Заверши фразой что эксперт свяжется в
ближайшее время. Пиши по-русски, просто и по-человечески,
без канцелярита.
```

### 1.4 Дополнительные команды бота

| Команда / слово | Что делает бот |
|---|---|
| `/start` или «Привет» | Короткое приветствие + инструкция написать ПРОВЕРКА |
| «ПРОВЕРКА» (любой регистр) | Запускает основной сценарий |
| Любой другой текст | «Напишите слово ПРОВЕРКА — и мы разберём вашу ситуацию бесплатно» |

### 1.5 Технический стек бота

| Компонент | Решение | Стоимость |
|---|---|---|
| Платформа | MAX Bot API (официальный) | Бесплатно |
| Язык | Python 3.11+ | Бесплатно |
| Библиотека MAX | `max-botapi-python` (официальная, GitHub) | Бесплатно |
| ИИ | Google Gemini 2.5 Flash API | Бесплатно (500 запросов/день) |
| Библиотека Gemini | `google-generativeai` | Бесплатно |
| Хостинг | Railway (бесплатный tier) | Бесплатно |
| Хранение состояния диалога | Словарь в памяти (достаточно для MVP) | Бесплатно |

Groq не используем — были проблемы с подключением. Единственный ИИ — Gemini.

### 1.6 Что нужно получить перед разработкой

- [ ] Токен MAX-бота — создаётся через `@BotMaster` в приложении MAX
- [ ] API-ключ Gemini — получить на [aistudio.google.com](https://aistudio.google.com), бесплатно
- [ ] MAX ID Яны — куда бот шлёт уведомления о новых запросах
- [ ] Аккаунт на Railway — для хостинга бота
- [ ] api_id и api_hash от Telegram — для парсера каналов (бесплатно на my.telegram.org)

### 1.7 Структура файлов бота

```
bot/
├── main.py          — точка входа, запуск бота
├── handlers.py      — логика сценария (ПРОВЕРКА, вопросы, ответы)
├── gemini.py        — запрос к Gemini API
├── notifier.py      — уведомление эксперту
├── config.py        — токены и настройки (через переменные окружения)
└── requirements.txt — зависимости
```

### 1.8 Переменные окружения (секреты, не хранятся в коде)

```
MAX_BOT_TOKEN=...        токен бота из @BotMaster
GEMINI_API_KEY=...       ключ из Google AI Studio
EXPERT_MAX_ID=...        MAX ID эксперта для уведомлений
```

## Часть 2. Канал в MAX

### 2.1 Настройка шапки канала

Название канала:
```
Перепланировка | Проверка рисков
```

Описание (шапка):
```
Перепланировка без штрафов и красных линий.
Проверяем риски до ремонта, сделки или ипотеки.
Инженеры СРО и НОПРИЗ · По всей России удалённо

Напишите слово ПРОВЕРКА — разберём ваш случай бесплатно.
```

Ключевые слова в описании для поиска внутри MAX:
`перепланировка`, `согласование`, `узаконить`, `БТИ`, `жилинспекция`

### 2.2 Постинг по расписанию

**Инструмент:** нативный планировщик MAX (удержание кнопки отправки → «Запланировать»)

**Для кросспостинга MAX + Telegram:** бот posting.su (бесплатно, подключается внутри MAX)

**Ритм публикаций:**

| День | Тип поста |
|---|---|
| Понедельник | Образовательный (что можно/нельзя, закон) |
| Среда | Кейс или разбор ситуации |
| Пятница | Интерактив (опрос «можно или нельзя?») |

Минимум — 3 поста в неделю. Этого достаточно для алгоритма MAX на старте.

**Подготовка контента:** пакет из 6–10 постов готовится раз в 2 недели вместе с Клодом. Время — 1–2 часа раз в 2 недели.

### 2.3 Первые 5 постов (порядок публикации)

1. Якорный — кто мы (с фото Сергея и Яны)
2. «Узнай себя» — триггер-ситуации
3. «Можно или нельзя?» — опрос
4. Кейс — реальная ситуация с результатом
5. «Что никогда не согласуют» — польза и предупреждение

## Часть 3. Точки входа трафика

### 3.1 Яндекс.Карты

- Открыть карточку организации в Яндекс Бизнес
- В раздел «Ссылки» добавить ссылку на канал MAX
- Описание обновить: упомянуть «бесплатная проверка рисков перепланировки»
- Настройка: 1 раз, работает постоянно

### 3.2 2ГИС

- Войти в кабинет 2ГИС
- В карточку добавить ссылку на канал MAX
- Проверить что телефон и мессенджеры актуальны
- Настройка: 1 раз, работает постоянно

### 3.3 Авито

- Создать шаблон ответа со ссылкой на канал:
  «Кстати, у нас есть канал в MAX — там отвечаем на вопросы по перепланировкам бесплатно: [ссылка]»
- Вставлять шаблон в каждый ответ одним нажатием
- Настройка: 5 минут, работает постоянно

### 3.4 SEO внутри MAX

- Название канала содержит слово «перепланировка»
- Описание содержит: перепланировка, согласование, узаконить, БТИ
- Этого достаточно для поиска внутри приложения

## Часть 4. Аналитика

Встроенная аналитика MAX — бесплатно, в admin-панели канала:
- охват каждого поста
- прирост подписчиков
- просмотры

Раз в неделю просматривать дашборд: что зашло, что нет.
Ориентир: посты с охватом выше среднего — повторять тему и формат.
