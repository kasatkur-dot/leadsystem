## 4. Что смотреть Agent 5

Agent 5 должен связывать:

```text
content_id -> channel -> CTA -> lead_count -> qualified_count -> deal_count -> revenue -> decision
```

Минимальные поля:

| Поле | Зачем |
|---|---|
| `content_id` | Понять, какой пост дал сигнал |
| `channel` | Понять, где сработало |
| `cta` | Понять, какой призыв дал вопросы |
| `lead_count` | Сколько лидов пришло |
| `qualified_count` | Сколько лидов реально целевые |
| `deal_count` | Сколько дошло до сделки |
| `revenue_rub` | Сколько денег принесло |
| `decision` | Что делать дальше |

## 6. Решение на следующий месяц

| Решение | Что попадает сюда |
|---|---|
| `scale` | Тема или формат дали лиды, вопросы или сильный отклик |
| `keep` | Нормальный результат, продолжать осторожно |
| `rewrite` | Идея хорошая, но текст/формат слабый |
| `pause` | Тема не дала сигнал или не подходит аудитории |

## 8. Ограничения

- Не вставлять реальные клиентские данные.
- Не вставлять токены, webhook-ссылки, пароли или session-файлы.
- Не считать публикацию разрешённой только потому, что она есть в плане.
- PostMyPost и автопубликацию подключать только отдельным решением.
