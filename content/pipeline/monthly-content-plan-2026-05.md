## 5. Статусы

Текущий статус:

- `content-2026-05-001`: `draft`, создан только технический dry-run без API и без публикации.
- `content-2026-05-002`: `planned`.
- `content-2026-05-003`: `planned`.

Дальше двигаться только так:

```text
planned -> draft -> review -> needs_changes / approved
```

Пока не используем:

```text

```

Почему: реальная публикация и автопостинг не разрешены на этом шаге.

## 7. Проверка перед созданием черновиков

Перед запуском dry-run Agent 4 проверить:

- нет персональных данных;
- нет обязательной формы захвата контакта;
- нет одинакового текста во все каналы;
- есть понятный следующий шаг;
- тема связана с гипотезой месяца;
- есть хотя бы 2-3 research source из watchlist или внутренних материалов;
- чужие тексты и визуалы не копируются, используем только паттерны и выводы;
- PostMyPost и автопубликация не используются;
- публикация не выполняется.
