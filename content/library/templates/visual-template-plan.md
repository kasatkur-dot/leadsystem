# План шаблонов визуалов

## Карточки

- `card-risk` - пост на боль и риск
- `card-can-cannot` - формат `можно / нельзя`
- `card-3-risks` - три риска
- `card-case` - кейс `было / стало / результат`
- `card-cta` - карточка с CTA `ПРОВЕРКА`
- `card-news` - новостной разбор
- `card-doc` - документ, план, БТИ, ЕГРН, красные линии
- `card-trust-process` - спокойный процесс работы компании шаг за шагом

## Видео

- `video-can-cannot`
- `video-3-risks`
- `video-case-30s`

## Что нужно у каждого шаблона

- назначение;
- размер;
- композиция;
- тип текста;
- допустимые цвета;
- место для CTA;
- можно ли использовать реальные материалы;
- можно ли усиливать AI-графикой.
