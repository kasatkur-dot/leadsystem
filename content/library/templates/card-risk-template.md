# Шаблон карточки: Риск

## Для чего

Карточка для постов, где нужно быстро показать человеку, что ситуация может быть опасной.

## Структура

1. Крупный заголовок с болью.
2. 3-5 коротких признаков риска.
3. Один вывод простыми словами.
4. CTA `ПРОВЕРКА`.

## Визуальные правила

- светлый фон;
- графитовый текст;
- синий акцент;
- красный только на словах `риск`, `стоп`, `внимание`;
- один экран = одна мысль;
- не больше 25-35 слов на карточку.
