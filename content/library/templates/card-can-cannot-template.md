# Шаблон карточки: Можно / нельзя

## Для чего

Карточка для формата `можно / нельзя / зависит`.

## Структура

1. Один конкретный вопрос.
2. Короткий ответ.
3. Одно объяснение почему.
4. Если нужно, пометка `зависит от условий`.
5. CTA `ПРОВЕРКА`.

## Визуальные правила

- один вопрос на карточку;
- зеленый = можно;
- красный = нельзя;
- синий = зависит;
- не превращать в сложную таблицу.
