# Черновики постов

## Зачем нужна эта папка

Здесь лежат готовые или почти готовые тексты постов для канала.

## Как работаем

1. Берем тему из `content/ideas/idea-registry.csv`.
2. Пишем простой черновик поста.
3. Сразу отмечаем:
   - рубрику;
   - цель поста;
   - какой нужен визуал;
   - нужен ли норм-чек.
4. После проверки переносим пост в недельный план.

## Главное правило

Пост должен вести не просто к просмотру, а к следующему действию:

- человек узнал свою ситуацию;
- понял риск;
- написал `ПРОВЕРКА`.
